🎤 Siri API

Control your Windows PC from iOS Shortcuts and Siri

What is Siri API?

Siri API is a beginner-friendly Python Flask web API that runs on your Windows PC. It lets you trigger actions on your computer using voice commands on your iPhone or iPad through iOS Shortcuts and Siri.

Say "Hey Siri, run my script" and watch your PC respond in real-time.

Secure

API key authentication. Whitelist of safe actions only—no arbitrary shell commands.

Easy Setup

Works with iOS Shortcuts. Add a voice phrase in seconds.

Cross-Platform

Python-based. Run locally or expose to the internet with ngrok.

Extensible

Add your own endpoints. Control Notepad, Calculator, URLs, and more.

Quick Start (Windows)

1. Install Python

Download Python 3.8+ from python.org

2. Clone the Repository

git clone https://github.com/aUserOnAndroid/siri-api.git cd siri-api

3. Set Up Virtual Environment

python -m venv .venv .\.venv\Scripts\activate pip install -r requirements.txt

4. Configure Your API Key

copy .env.example .env

Open .env in Notepad and replace API_KEY=change_me with a secure key of your choice.

5. Run the Server

Choose one:

PowerShell: .\run.ps1 or: .\run.bat
✓ Your API is now running at http://localhost:5000

6. (Optional) Expose to the Internet

To test from your iPhone/iPad without being on the same Wi-Fi:

ngrok http 5000

Copy the HTTPS URL ngrok gives you. Use that URL in your Shortcut instead of localhost.

Available Endpoints

All requests require the API key as a header: X-API-Key: YOUR_KEY

Endpoint Method Description
/say GET Text-to-speech. Query: text=Your message
/run POST Run a safe action. Supports: open, launch, url
/note POST Create a note on your Desktop
/health GET Check if the server is running

Create Your First Shortcut

Step 1: Prepare Your Server

  • Start the Flask server on Windows (run.ps1 or run.bat)
  • Note your API key from .env
  • If testing remotely, run ngrok http 5000 and copy the HTTPS URL

Step 2: Open iOS Shortcuts

  • Open the Shortcuts app on your iPhone/iPad
  • Tap "+" to create a new Shortcut

Step 3: Add a URL Request

  • Search for and add "Get Contents of URL" action
  • Set the URL to: https://YOUR_SERVER/say?text=Hello%20World
  • Change Method to GET
  • Add Headers: X-API-Key = YOUR_KEY

Step 4: Add Siri Voice Command

  • In Shortcut settings, tap "Add to Siri"
  • Record a voice phrase like: "Run my script"
  • Save

Step 5: Test It!

Say "Hey Siri, run my script" and your PC will respond with text-to-speech.

✓ Congratulations! Your first Siri-enabled action is ready.

Security

⚠️ Keep your API key secret! Treat it like a password.

Key Features:

  • API Key Authentication: All endpoints require a valid API key in the X-API-Key header
  • Whitelisted Actions Only: The /run endpoint only accepts safe, pre-approved actions (open URLs, launch apps)
  • No Arbitrary Commands: You cannot run arbitrary shell commands through the API
  • Remote Access: When testing from the internet, use ngrok and rotate your API key if it's ever exposed

Best Practices:

  • Never commit your real API key to the repository
  • Use .env to store secrets locally
  • For internet access, prefer ngrok over direct port forwarding
  • Use a strong, random API key
  • Don't share your ngrok URL publicly

Troubleshooting

"Connection refused" error

  • Make sure the Flask server is running on Windows
  • Check that the server is listening on port 5000
  • Verify your iPhone/iPad can reach your PC (same Wi-Fi, or ngrok URL)

"401 Unauthorized" error

  • Check your API key in the X-API-Key header
  • Make sure it matches the API_KEY in your .env file

"Shortcut not working"

  • Test the URL in your Safari browser first
  • Make sure the "Get Contents of URL" action has the correct headers
  • Check the server logs for error messages

Remote access with ngrok not working

  • Restart ngrok: ngrok http 5000
  • Use the new HTTPS URL in your Shortcut
  • Make sure your API key header is included
